# Contabilidad

Aplicacion web estatica para cargar facturas mensuales y consultar resumenes trimestrales.

## Como usar

Abre `index.html` en el navegador.

La app permite:

- Entrar desde el boton `CONTABILIDAD`.
- Cargar importes por meses desde `CARGAR FACTURAS`.
- Calcular automaticamente `TOTAL`, `IVA`, `TOTAL FACTURA`, `GASTOS` e `INGRESOS`.
- Consultar resumenes en `TRIMESTRAL`.
- Guardar los datos en el navegador con `localStorage`.

## Publicar en GitHub Pages

1. Sube estos archivos al repositorio:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
   - `.nojekyll`
2. En GitHub, entra en `Settings`.
3. Abre `Pages`.
4. En `Build and deployment`, elige `Deploy from a branch`.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Pulsa `Save`.

GitHub publicara la app como una pagina web.
